# 얼굴인식 api를 활용한 도어락
<br>

## Overview 

파이썬의 얼굴인식 api를 라즈베리에서 활용하여 도어락 기능을 구현하고   
아마존의 클라우드서버를 활용하여 안드로이드 어플리케이션에서   
사진 관리 및 원격 출입을 구현하였습니다.

**📆 프로젝트 기간 : 2023.04 ~ 2023.10**

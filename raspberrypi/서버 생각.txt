파이썬에서 소켓서버 열고 변수선언 쪽에 서버 생성
초음파 while문 안에서 소켓 생성하고 입력대기
입력 들어오면 string 받아서 모터 돌리고 안드로이드로 열렸다고 string 보내기
sleep후 모터 다시 돌리고 replay==1하면 continue 실행됨
얼굴인식 중이면 소켓연결 실패하도록 설정해야함

listen에서 코드가 멈추고 accept가 되면 다시 돌아가는데..
이러면 
1. aws에서 소켓열어서 파이썬이랑 항상 통신시키고 안드로이드에서 버튼 클릭시 aws랑 소켓통신해서 aws에서 파이썬으로 string 전송
2. 파이썬은 계속 클라이언트 접속 요청하면서 대기 안드로이드에서 버튼 클릭시 서버 열면서 연결
3. 파이썬에서 쓰레드로 서버 실행(되는지 모름)
4.settimeout 3초정도 설정해서 연결안되면 에러띄우고 다시 while문 돌리는 방식


안드로이드는 버튼 클릭하면 소켓 생성 서버랑 연결하고 string 보내기
안보내지면 서버 업데이트 or 얼굴인식 토스트 출력
보내지면 파이썬으로부터 string 받아서 열렸다고 토스트 출력
